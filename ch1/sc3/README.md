# Scene 3 - Two Characters Animation

## Required Assets

Place these files in this folder:

### Background
- `bg.png` - Background image

### Character 1 (Left Side)
- `01.png` - Frame 1
- `02.png` - Frame 2
- `03.png` - Frame 3

### Character 2 (Right Side)
- `1.png` - Frame 1
- `2.png` - Frame 2
- `3.png` - Frame 3
- `4.png` - Frame 4
- `5.png` - Frame 5

## Animation Sequence

**Character 1:** Plays once: 01 → 02 → 03

**Character 2:**
- First play: 1 → 2 → 3 → 4 → 5
- Repeat frames 2-5 twice more: 2 → 3 → 4 → 5 (×2)
- Total sequence: 1, 2, 3, 4, 5, 2, 3, 4, 5, 2, 3, 4, 5
