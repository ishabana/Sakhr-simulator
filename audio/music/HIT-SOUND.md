# Hit Sound Effect for Space Invaders Scene

## What You Need

A short hit/explosion sound effect for when bullets hit invaders in sc2.

## Specifications

- **File name**: `hit.mp3`
- **Duration**: 0.1 - 0.3 seconds (very short)
- **Style**: MSX-style beep or explosion
- **Volume**: Will be played at 50% volume

## Where to Get

### Option 1: Generate Online
- [Sfxr](https://sfxr.me/) - Classic 8-bit sound generator
  - Select "Explosion" or "Hit/Hurt"
  - Export as WAV, convert to MP3

### Option 2: Use Beepbox
- [Beepbox.co](https://www.beepbox.co)
- Create a very short (1 note) explosion sound
- Export as MP3

### Option 3: Find Royalty-Free
- Search "8-bit explosion sound effect free"
- Download and rename to `hit.mp3`

### Option 4: Use Any Short Beep
- Any short sound will work for testing
- Just needs to be quick and noticeable

## Installation

Place the file here:
```
src/animation/audio/music/hit.mp3
```

The game will automatically use it when available. If not present, game works silently.
