# ROM Cover Images

Place your custom ROM cover images in this folder.

## Required Image Files

```
abc-arabic.png       → 600×400px (ABC العربية)
believe-or-not.png   → 600×400px (صدق أو لا تصدق)
double-face.png      → 600×400px (الوجه المزدوج)
holy-quran.png       → 600×400px (القرآن الكريم)
ibn-sina.png         → 600×400px (ابن سينا)
```

## Image Guidelines

- **Format**: PNG (preferred) or JPG
- **Size**: 600×400px or similar 3:2 aspect ratio
- **Style**: Retro MSX aesthetic, screenshots, or box art
- **Colors**: Use MSX color palette when possible
- **Text**: Arabic text for authenticity

## Fallback

If image is missing, a 📦 placeholder icon will appear automatically.

## Tips for Creating Images

1. **Screenshots**: Capture actual MSX program screens
2. **Box Art**: Scan or recreate Sakhr box covers
3. **Custom Graphics**: Design in MSX color palette
4. **Pixel Art**: 8-bit style graphics work best
5. **Arabic Text**: Use FF Sakhr or similar MSX fonts

## Example Sources

- MSX screenshots from actual programs
- Scanned Sakhr AX-370 cartridge boxes
- Recreated logos/graphics in pixel art style
- Archive.org MSX software library
